## üîß Troubleshooting

**Photo not showing?**
- Make sure file is named exactly: `profile.jpg`
- Save it in the same folder as `index.html`
- Check the file format (JPG, PNG, or WebP)

**Animations not smooth?**
- Try a different browser
- Check if JavaScript is enabled
- Refresh the page

**Form not working?**
- Make sure Flask backend is running
- Check email configuration in `app.py`
